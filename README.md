# Sales-Performance-Dashboard
Task1, Sales Performance Dashboard using Excel for Elevvo Data analyst Internship

## Project Overview
This dashboard was created as part of my internship to analyze the Superstore Sales Dataset. It provides insights into revenue trends, monthly growth, and regional performance.

## Features
- **Total Revenue & Units Sold:** Key business metrics at a glance.
- **Monthly Trends:** Visualized via Line Charts to identify seasonality.
- **Interactivity:** Slicers for 'Region' and 'Category' to filter data dynamically.
- **Calculations:** Used Pivot Tables to calculate % Month-over-Month (MoM) change.

## Tools Used
- Microsoft Excel (Pivot Tables, Slicers, Data Visualization)
- Dataset: Superstore Sales Dataset
